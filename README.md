# ⚙️ Projeto

Reprodução da interface da Twitch utilizando ReactJS e Tailwind CSS. Com implementação do carrossel infinito, tema escuro, grid responsivo e animações.

# ✈️ Tecnologias

- ReactJS
- Tailwind CSS
